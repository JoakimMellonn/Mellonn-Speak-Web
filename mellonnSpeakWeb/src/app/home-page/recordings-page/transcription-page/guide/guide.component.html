I'm not new though...

Editing what's being said

Say what?! If it wasn't what was said in the recording, you can always change it!

This is done easily by clicking the bubble and changing the text. When done, you can press enter or the save button.

Editing who says what

If the AI didn't get it quite right, you can always change who's talking when and saying what.

This is done by selecting the text you want to change, and selecting the appropriate speaker.

Controlling the audio

You can always control the audio, with the media controls in the bottom of the screen.

When pressing a bubble, the given part of the audio is selected in the progress bar. Double anywhere outside of the bubble, to reset this selection.

Changing the labels

At any time you can give the participants labels, so you know who said what!

Defining who's an interviewer or interviewee, changes who gets the green bubbles. You can use the play/shuffle buttons, to be sure who's who.

Go back to older versions

Something happened that shouldn't happen, and you need to go back in time?

Going back in time I can't help with, but going back in versions I can! Just go into the version history via the three dot menu, and the choose the version you wanna go back to! The top version is the default version (the one created by the AI), just under there is the newer versions, starting with the current one.

Export transcription

When you're all done, you can export the transcription as a Word-document (DOCX).

This is done in the three dot menu, and then "Download as DOCX".

You're all done learning now, have fun and save some time!
